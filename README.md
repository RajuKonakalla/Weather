# Weather App by Raju Konakalla

## Project Overview

This weather application provides current weather information, forecasts, and other weather-related insights. It is designed to be user-friendly and visually appealing, leveraging modern web technologies.

## Technologies Used

- Vite: Fast frontend build tool and development server.
- TypeScript: Typed superset of JavaScript for better code quality and maintainability.
- React: Popular JavaScript library for building user interfaces.
- shadcn-ui: Component library built on top of Radix UI and Tailwind CSS.
- Tailwind CSS: Utility-first CSS framework for rapid UI development.

## Required Dependencies

- Node.js (version 16 or higher recommended)
- npm (comes with Node.js)
- Vite (installed as a dev dependency)
- React and React DOM
- TypeScript
- Tailwind CSS
- shadcn-ui components and Radix UI primitives

These dependencies are managed via the package.json and installed with `npm install`.

## How to Clone and Use

1. Clone the repository:
```sh
git clone https://github.com/RajuKonakalla/Weather.git
cd Weather
```

2. Install dependencies:
```sh
npm install
```

3. Run the development server:
```sh
npm run dev
```

4. Open your browser and navigate to:
```
http://localhost:8081/
```
